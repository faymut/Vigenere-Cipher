# Vignere-Cipher
Vignere Cipher Encryptor

A lightweight Vignere cipher tool that supports English, Arabic, digits, and symbols. Built for tactical encryption and decryption using a custom key-based shift.
Features

    Encrypts and decrypts:

        English (uppercase + lowercase)

        Arabic letters

        Digits (0–9)

        Symbols and punctuation

    Key-based Caesar shift using character code sum

    No external libraries — pure JavaScript

    Designed for browser use with minimal HTML

How It Works

    Enter your text and key.

    Click Encrypt to shift characters forward.

    Click Decrypt
